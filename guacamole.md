# Guacamole (remote desktop)

You can log in to the Guacamole remote desktop service at
http://symmetry.pi.local/guacamole/#/

[guacamole login screenshot](/assets/guacamole-login.png)

After logging in, you will get the main menu screen.  Select the option
"Connect to Remote Desktop (VNC), launching one if necessary", highlighted in green.

[guacamole main menu](/assets/guacamole-menu.png)
